# ascorbic/skills

A Claude Code plugin marketplace for web development.

## Installation

```bash
/plugin marketplace add ascorbic/skills
/plugin install astro@ascorbic-skills
```

## Plugins

### astro-skills

Astro development tools including:

**Skills:**
- `astro-v6-upgrade` - Guide for upgrading from Astro v5 to v6 with progressive disclosure

**MCP Servers:**
- `astro-docs` - Real-time access to Astro documentation
